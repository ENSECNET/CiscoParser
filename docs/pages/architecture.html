CiscoParserENSECNET · network docs

Architecture

FastAPI backend + static frontend, packaged as Docker Compose on a minimal LXC.

.cfg ──► parser.py ──► transform.py ──► netbox_push.py ──► NetBox

Modules

  • parser.py — CiscoConfParse2 wrapper. Extracts native objects as structured fields, everything else as textual config-context blocks.
  • transform.py — reshapes raw parser output into the canonical NetBox model. Without this layer the push produces an empty device.
  • netbox_push.py — pynetbox. Auto-creates prerequisites and sets the three required device fields (device_type / role / site).

Why this base image choice

Alpine for the Docker images (real size win, ~50–60 MB backend). Debian 12 minimal for the LXC host, because Docker runs there without the cgroupv2 / OpenRC friction you hit running Docker on an Alpine LXC.
