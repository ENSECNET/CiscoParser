Web edition

A public, zero-install converter — for when you just want the model, without a server.

The CiscoParser Web repo runs the same parser as a Cloudflare Worker (pure JavaScript). Paste a config, get the model, download it.

paste config ──► parse (Worker) ──► model JSON ├─► Download JSON └─► Download import.py (run it yourself)

Why it does not push

NetBox lives on a private network and a token is sensitive. The web edition never pushes and never sees your token. Instead it returns the model as JSON plus a self-contained import.py you run on a host that can reach your own NetBox, with your own credentials.

Same model shape as the on-prem edition, so the JSON and the generated script are interchangeable between the two.

Webová edícia

Verejný konverter bez inštalácie — keď len chceš model, bez servera.

Repo CiscoParser Web beží na rovnakom parseri ako Cloudflare Worker (čisté JavaScript). Vložíš config, dostaneš model, stiahneš ho.

vlož config ──► parse (Worker) ──► model JSON ├─► Stiahni JSON └─► Stiahni import.py (spustíš sám)

Prečo nepushuje

NetBox žije v privátnej sieti a token je citlivý. Webová edícia nikdy nepushuje a nikdy nevidí tvoj token. Namiesto toho vráti model ako JSON plus samostatný import.py, ktorý spustíš na hoste s dosahom na tvoj vlastný NetBox, s tvojimi vlastnými prihlasovacími údajmi.

Rovnaký tvar modelu ako on-prem edícia, takže JSON aj vygenerovaný skript sú medzi nimi zameniteľné.
