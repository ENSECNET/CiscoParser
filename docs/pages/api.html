API

The backend exposes a small REST surface behind Nginx at /api.

MethodEndpointPurpose
GET/api/healthliveness
POST/api/test-connectionverify NetBox URL + token
POST/api/parseupload .cfg, return model
POST/api/parse-textparse raw text body
POST/api/pushpush model to NetBox

Example — parse then push

curl -s -X POST http://HOST:8080/api/parse-text \
  -H 'content-type: application/json' \
  -d '{"config":"hostname CORE-RTR-01\n!\ninterface Gi0/0/0\n ip address 198.51.100.2 255.255.255.0"}'
